# MaHalTaNa-A-Cloud-Based-Local-Event-Discovery-Platform-for-Oriental-Mindoro

Capstone Project

MaHalTaNa is a cloud-based local event discovery platform developed to improve event discoverability, management, and promotion in Oriental Mindoro. 
The system provides a centralized platform that supports local cultural and economic initiatives while promoting digital innovation and community engagement.

**My Role**

-UI/UX Design
-Research and Documentation

**Technologies Used**

-Laravel
-Bootstrap 5
-MariaDB
-Laragon

This project was developed following the Web Development Life Cycle (WDLC) to deliver an efficient, scalable, and accessible web application. Future enhancements include real-time notifications and an integrated chat/messaging feature to further improve user engagement.
